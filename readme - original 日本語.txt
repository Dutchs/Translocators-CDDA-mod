◆translocator MOD

◇概要
	遠距離を瞬間移動できるアイテムとそれに伴うあれやこれやを追加するMODです。
	瞬間移動の対象は「プレイヤーキャラクターのみ」です。

◇注意事項
	作成・動作確認はCDDA開発版#8124で行っていますが
	これ以外のバージョンでの動作を保証する物ではありません。
	
	Luaを使用したMODです。
	不定のタイミングで使用不能になる可能性があります。
	
	MODの導入によりセーブデータに深刻なエラーや損害が発生する可能性があります。
	MOD導入は自己責任で行って下さい。
	
	処理の特性上、CTD(ゲームが落ちる)の可能性が常に付き纏います。
	定期的なセーブの実行やバックアップ等の対策を行う事を推奨します。
	
◇導入方法
	1.[translocator]フォルダをCDDAの[Mods]フォルダにコピー
	2.世界生成のMODリストの[基本]タブから【追加 - 長距離転送】を選択
	3.遊ぶ

◇レシピ
	追加アイテムは電子工学スキルLv8で自動習得します。
	また、追加される建造物は製作スキルLv7電子工学スキルLv8で完成させる事が出来ます。

◇MOD詳細
	追加されるアイテム
		[TRS]瞬間転送装置
			使用すると
			◇登録先へ転送する
				登録された転送先へチャージを消費して瞬間移動します。
				消費するチャージ量はゲートへの転送の場合「6」
				登録した位置への転送の場合「12」です。
			◇転送先を登録する
				転送先の位置としてアイテムに位置情報を記録します。
				チャージの消費はしません。
			◇転送先を解除する
				記録された転送先の情報をアイテムから削除します。
				チャージの消費はしません。
			◇転送ゲートとリンクする
				転送ゲート(後述)とアイテムをリンクさせます。
				転送位置の登録とは別枠で1つ転送先としてゲート位置を記録します。
				チャージの消費はしません。
			◇低精度転送
				マップ画面で座標を指定してその位置へ瞬間移動します。
				チャージを「24」消費します。
				ただし転送先の安全は保証されません(＊＊いしのなかにいる！！＊＊とか起こります)
				また後述の転送事故の起きる確率が倍増します。
			を行えるアイテムです。
			転送位置は一つのアイテムに対し4つまで登録できます。
			
			アイテムのチャージは最大24で消費したチャージは1時間に1ポイント自動で回復します。
	
	追加される地形
		転送ゲート基部
			転送ゲートを設置する為の建設の1段階目です。
			建築メニューの建設タブから建築できます。
			この状態では移動の邪魔になりアイテムが置けないだけのお邪魔地形です。
		
		転送ゲート
			転送の目標地点になる地形です。
			転送ゲート基部から更に建設する事でこの地形へと変わります。
			建築メニューの建設タブから建築できます。
			建築パターンが2種類ありますが、どちらで建設しても変わりありません。
			
			この地形の上に立って上記の追加アイテムを使用しリンクさせることで転送先として登録されます。
			ゲートへの転送位置の登録は4つの登録枠を消費せずに登録が可能です。
			
			また、ゲートとのリンクは転送事故(後述)による転送先登録消失の影響を受けません。
	
	転送事故
		瞬間移動をすると低確率で悪い事が起きます。
		また、ゲートへの転送の場合、発生確率が半分ほどになり
		低精度転送の場合、発生確率が倍になります。
		
			転送酔い(軽)
				1時間程「Teleglow」を発症します。
				「なんだか気分が悪い。」
				
				発生確率：5%程
			
			転送酔い(重)
				6時間程「Teleglow」を発症します。
				「とても気分が悪い。」
				
				発生確率：3%程
			
			全身負傷
				全身に1～10ポイントのダメージを受けます。
				「痛い！！」
				
				＊＊このダメージで死ぬ事もあります＊＊
				緊急時の転送等は注意して下さい。
				
				発生確率：1%程
			
			転送装置破損
				転送装置に登録された転送先情報が「全て」消失します。
				「転送装置が煙を吹いている！」
				ただし、ゲートとのリンクは失われません。
				
				要はゲート以外への転送が再度登録するまで出来なくなりますよ、と。
				
				発生確率：0.2%程
		
		これらの転送事故を起こしたくない場合「preload.lua」をテキストエディタで開いて
		一行目の
		local enable_penalty = true
		を
		local enable_penalty = false
		に書き直して下さい。
